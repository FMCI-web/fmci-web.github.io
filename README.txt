FMCI Web App website package
============================

Contents
--------
index.html                  Updated home page
modules/*.html              Seven detailed Learn more pages
assets/module-pages.css     Shared styling for module pages

Launch URL
----------
https://unshaven-energy-verify.ngrok-free.dev/webapps/home/session.html?app=WuWebAppDesignerWeb2

Use
---
1. Keep the folders and files together.
2. Open index.html locally to preview.
3. Upload the complete FMCI_Web_App_site folder to your web host.
4. The Learn more cards use relative links, so preserve the modules folder.

Built from the detailed instruction text in WuWebAppDesignerCore.m and the
module routines supplied in the attached MATLAB app archive.
